# 台灣捷運動態圖

以捷運公司標準路網圖為靈感的全台捷運動態網站，針對手機瀏覽設計。

公開網址：

<https://gt01866xa61.github.io/taiwan-mrt-live/>

## 目前涵蓋

- 北北桃：台北捷運、新北捷運、桃園機場捷運
- 台中：台中捷運綠線
- 高雄：高雄捷運紅線、橘線與環狀輕軌
- 共 18 條路線、258 個車站

## 資料說明

畫面中的列車不是逐車 GPS 定位。網站以台北時間推算列車位置，北捷文湖、淡水信義、
松山新店、中和新蘆、板南與環狀線優先採用北捷公開的逐站行駛／停靠秒數（資料生效日
2025-06-03）；其餘路線使用公開班距、站序與平均旅速模型。北捷官方事件校正只會在
路線、方向、終點與既有班次可唯一辨識時套用，且以 0.25–2 倍的單向速度平順收斂，
不會把車點拉回上一站或瞬間跳躍。

北捷文湖、淡水信義、松山新店、中和新蘆與板南線另會讀取臺北市資料大平臺的
「列車進入月台」事件。事件含部分短折返終點（例如大安、台電大樓、亞東醫院），
網站只把它當成既有班次的進站錨點，不額外新增一批列車。事件本身沒有列車編號、
完整列車清單、ETA 或站間座標，因此仍不是逐車連續定位；官方資料延遲、歧義或中斷
時，網站會自動保留原本的班表推估。

網站首頁會常駐揭露位置推估語義，並在北捷資料可用時顯示官方到站事件的新鮮度。
列車數代表目前畫面所選路線的推估數量，不代表營運公司公布的實際在役列車數。

## 使用方式

- 北北桃預設聚焦台北核心路網，可從底部「路線」切換全部路網、新北輕軌或機場捷運。
- 台中與高雄同樣可從路線選單逐線開關。
- 列車採非對稱圓弧車頭，車頭方向即為推估行進方向。
- 所有路線都被隱藏時，可用畫面中央的按鈕一鍵恢復。

## 本機預覽

這是零建置步驟的單頁網站，可直接開啟 `index.html`，或在專案根目錄執行：

```bash
python -m http.server 8000
```

再開啟 <http://localhost:8000/>。

## 驗證

修改後至少確認：

1. JavaScript 語法可解析。
2. 北北桃、台中、高雄三個分頁可切換。
3. 手機尺寸可拖曳、縮放、點選列車與車站。
4. 頁面仍清楚揭露「依班距推算，非車載 GPS」。
5. 路線選單的群組、逐線開關與全部恢復功能正常。

純 JavaScript 回歸測試可用 Node.js 執行：

```bash
node --test tests/*.test.mjs
```
