# 台灣捷運動態圖

以捷運公司標準路網圖為靈感的全台捷運動態網站，針對手機瀏覽設計。

公開網址：

<https://gt01866xa61.github.io/taiwan-mrt-live/>

## 目前涵蓋

- 北北桃：台北捷運、新北捷運、桃園機場捷運
- 台中：台中捷運綠線
- 高雄：高雄捷運紅線、橘線與環狀輕軌
- 共 18 條路線、258 個車站

## 資料說明

畫面中的列車不是逐車 GPS 定位。網站依公開的營運時間、班距、站序與平均旅速，
以台北時間推算列車位置；所有訪客在相同時間會看到一致的模擬結果。

網站首頁會常駐顯示「班表推估・非 GPS」，列車數代表目前畫面所選路線的推估數量，
不代表營運公司公布的實際在役列車數。

## 使用方式

- 北北桃預設聚焦台北核心路網，可從底部「路線」切換全部路網、新北輕軌或機場捷運。
- 台中與高雄同樣可從路線選單逐線開關。
- 列車採非對稱圓弧車頭，車頭方向即為推估行進方向。
- 所有路線都被隱藏時，可用畫面中央的按鈕一鍵恢復。

## 本機預覽

這是零建置步驟的單頁網站，可直接開啟 `index.html`，或在專案根目錄執行：

```bash
python -m http.server 8000
```

再開啟 <http://localhost:8000/>。
